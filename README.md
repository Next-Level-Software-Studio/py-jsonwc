# py-jsonwc


py-jsonwc is a Python package for processing JSON documents with various types os comments, and other features.
Este projeto é um fork de [commentjson](https://github.com/vaidik/commentjson).
Créditos da equipe do commentjson.

## Installation:
### Bit-OS
```
emerge --sync
emerge dev-python/py-jsonwc
```
## Tests
```
python setup.py test
```
